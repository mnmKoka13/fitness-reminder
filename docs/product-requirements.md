# プロダクト要求定義書

## プロダクトビジョンと目的

運動を習慣化したいが「忘れる」「めんどくさい」「動画を探す手間」という3つの障壁に阻まれているユーザーのために、リマインダーと動画ショートカットを一体化したiOSアプリを提供する。

## ターゲットユーザー

- 自分自身（個人利用）
- InstagramやYouTubeの短い運動動画を参考に自宅で運動している
- 運動しようという意欲はあるが、行動に移すまでの摩擦が大きい

## 課題とニーズ

| 課題 | ニーズ |
|------|--------|
| 運動をつい忘れてしまう | 毎日決まった時間に通知してほしい |
| めんどくさくて先延ばしにしてしまう | 通知から即座に運動動画にアクセスできてほしい |
| 運動のたびに動画を探すのが大変 | よく使う動画をあらかじめ登録しておきたい |

## 主要な機能一覧

### MVP（プロトタイプ）
1. **毎日リマインダー** — 毎日1回、固定時間にローカル通知を送信
2. **動画リスト表示** — 登録済みの動画URLをアプリ起動直後に一覧表示
3. **動画を開く** — タップするとInstagramまたはYouTubeアプリで動画を開く
4. **動画の管理** — URLのペーストによる追加、削除、並び替え

### 将来機能（スコープ外）
- 動画への名前・メモ付け
- 運動ログ（実施記録）
- 曜日・時間の柔軟なスケジュール設定
- 共有シートからの動画登録

## 成功の定義

- 通知が毎日届き、タップして5秒以内に動画を開始できる
- 動画の追加・削除・並び替えが直感的に操作できる

## ユーザーストーリー

| # | ストーリー |
|---|-----------|
| 1 | 運動する時間になったら通知が届き、アプリを開いて動画をすぐ開始できる |
| 2 | 新しいお気に入り動画を見つけたときに、URLをコピーしてアプリに登録できる |
| 3 | 飽きた動画を削除したり、よく使う動画を上に並び替えできる |

## 受け入れ条件

- [ ] 毎日設定した時間にローカル通知が届く
- [ ] 通知タップでアプリが起動し、動画リストが即座に表示される
- [ ] 動画URLを貼り付けて登録できる
- [ ] 登録した動画をタップするとInstagram/YouTubeアプリが開く
- [ ] 動画を削除できる（スワイプ or 編集モード）
- [ ] 動画を並び替えできる（ドラッグ&ドロップ）
- [ ] 登録動画数の上限なし（数本程度の想定だが制限は設けない）

## 機能要件

### リマインダー
- ローカル通知（サーバー不要）
- 通知時刻はアプリ内で設定可能
- 通知文言は固定（「運動の時間です！」）

### 動画リスト
- 起動画面 = 動画リスト画面
- InstagramおよびYouTubeのURLに対応
- URLの形式バリデーションを行う
- データはデバイスローカルに永続保存

## 非機能要件

- **プラットフォーム**: iOS 17以上
- **ネットワーク**: 不要（ローカルストレージのみ）
- **プライバシー**: 外部サーバーへのデータ送信なし
- **シンプルさ**: 操作ステップを最小限に抑える
