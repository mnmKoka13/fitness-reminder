# 動画リストのフォルダ分け機能 要求内容

## 概要

動画リストを「フォルダ」で分類できるようにする。動画一覧画面内にフォルダ名のタブを設け、タブを切り替えることで分類ごとの動画を表示できるようにする。鍛えたい部位など、目的別に動画を整理したいというニーズに応える。

## 実装する機能

### 1. フォルダの作成・編集・削除

- 動画一覧画面から直接、フォルダ（タブ）の作成・名称編集・削除を行える
- 「デフォルト」フォルダを特別なフォルダとして常に1つ用意する
　- アプリ初回起動時、およびフォルダ導入前からの既存ユーザーのアップデート時に自動生成する
　- 「デフォルト」フォルダは削除できない（フォルダが0件の状態を作らないため）

### 2. 「すべて」タブ

- 動画一覧画面の先頭に「すべて」タブを常設する
- 「すべて」タブでは、フォルダをまたいで全動画をまとめて表示する

### 3. 動画のフォルダ割り当て

- 動画追加画面（AddVideoView）で、追加先のフォルダを選択してから保存する
- 一度追加した動画を後から別フォルダへ移動する機能は今回のスコープ外とする

### 4. 動画登録上限

- 登録上限（最大10件）は、フォルダ単位で適用する（フォルダごとに最大10件、アプリ全体としての合計上限は設けない）

### 5. 既存データの移行

- フォルダ導入前に登録済みの動画は、自動生成される「デフォルト」フォルダにすべて格納する
- 移行時点で10件を超えている場合も、動画を削除せずそのまま保持する（上限チェックは新規追加時のみ適用する）

## ユーザーストーリー

- 「胸トレ」「背中トレ」のようにフォルダを作って動画を整理し、タブを切り替えて見たい動画をすぐ探せる
- アップデート後も、これまで登録していた動画が消えずに残っている
- どのフォルダにも属さない動画も含めて全体を見返したいときは「すべて」タブで確認できる

## 受け入れ条件

- [x] 動画一覧画面でフォルダ（タブ）を新規作成できる
- [x] 動画一覧画面でフォルダ名を編集できる
- [x] 動画一覧画面でフォルダを削除できる（フォルダ内の動画も削除される。削除前に確認ダイアログを表示する）
- [x] 「デフォルト」フォルダは削除操作の対象にならない（削除UIを出さない、または削除不可の旨を伝える）
- [x] 「すべて」タブが常に先頭に表示され、全フォルダの動画がまとめて表示される
- [x] 動画追加時にフォルダを選択でき、選択したフォルダに動画が保存される
- [x] 各フォルダごとに動画が10件登録されると、そのフォルダでは＋ボタンが無効化され上限メッセージが表示される（他フォルダには影響しない）
- [x] アップデート後、既存の登録済み動画が「デフォルト」フォルダにすべて表示される
- [x] 既存動画が10件を超えていても削除されずに残る
- [x] アプリ再起動後もフォルダ構成・動画のフォルダ割り当てが保持される

## 制約事項

- 動画をフォルダ間で後から移動する機能は今回実装しない
- フォルダの並び替え機能は今回実装しない（作成順で表示）
- 外部ライブラリは使用しない
