# 初回実装 要求内容

## 概要

運動リマインダーiOSアプリのプロトタイプを実装する。

## 実装する機能

### 1. 動画リスト画面
- アプリ起動直後に動画リストを表示する
- 登録済みの動画URLを一覧表示する
- 動画セルをタップすると対応するアプリ（Instagram / YouTube）を開く
- 動画が0件の場合は「動画を追加してください」などの空状態を表示する

### 2. 動画管理
- URLを貼り付けて動画を追加できる
- Instagram / YouTube の URL であることをバリデーションする
- 不正なURLは保存できない（エラーメッセージを表示）
- スワイプまたは編集モードで動画を削除できる
- ドラッグ&ドロップで動画を並び替えできる
- 追加・削除・並び替えはローカルに永続保存される

### 3. リマインダー設定
- 設定画面で通知時刻（時・分）を設定できる
- 設定変更は即座に通知スケジュールに反映される
- 通知権限が未取得の場合は許可を求める
- 毎日設定時刻にローカル通知が届く

## ユーザーストーリー

- 通知が届いたらアプリを開いて、動画をタップしてすぐ運動を開始できる
- 新しいお気に入り動画のURLをコピーしてアプリに登録できる
- 飽きた動画を削除・並び替えできる

## 受け入れ条件

- [ ] アプリを起動すると動画リスト画面が表示される
- [ ] 動画が未登録の場合は空状態メッセージが表示される
- [ ] ＋ボタンをタップすると動画追加画面が開く
- [ ] 正しいInstagram/YouTubeのURLを入力して保存できる
- [ ] 不正なURLは保存できず、エラーが表示される
- [ ] 保存した動画がリストに表示される
- [ ] 動画セルをタップすると対応アプリ/Safariで開く
- [ ] スワイプで動画を削除できる
- [ ] ドラッグ&ドロップで並び替えができる
- [ ] アプリを再起動しても動画リストが保持されている
- [ ] 設定画面で通知時刻を変更できる
- [ ] 設定した時刻に毎日通知が届く

## 制約事項

- 外部ライブラリは使用しない
- iOS 17.0以上を対象とする
- ネットワーク通信は行わない
